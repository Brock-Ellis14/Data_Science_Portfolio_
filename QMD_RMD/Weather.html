Background

Dew point, the temperature at which air becomes saturated with moisture, is influenced by temperature and varies across regions due to factors like elevation and climate. This study explores how the relationship between temperature and dew point differs between Rexburg, Idaho and Yakima, Washington—two cities with distinct geographic and climatic characteristics.

Using daily 2024 weather data from the Global Summary of the Day (GSOD) dataset, we apply a multiple linear regression model to examine whether dew point responds differently to temperature in each location. The model includes an interaction term to allow both the slope and intercept of the relationship to vary by city.

This structure enables us to fit two separate regression lines—one for each location—and assess whether the rate at which dew point increases with temperature is consistent across regions or location-dependent.

Hypothesis

We begin by stating the Null and Alternative Hypotheses for each section of the multiple linear regression analysis:

\[ H_{0_1}: \beta_1 = 0 - \text{Temperature has no effect on dew point in Rexburg} \\ H_{A_1}: \beta_1 \ne 0 - \text{Temperature affects dew point in Rexburg} \\ \]

\[ H_{0_2}: \beta_2 = 0 - \text{Yakima has the same baseline dew point as Rexburg} \\ H_{A_2}: \beta_2 \ne 0 - \text{Yakima has a different baseline dew point} \\ \]

\[ H_{0_3}: \beta_3 = 0 - \text{The effect of temperature on dew point is the same in both locations} \\ H_{A_3}: \beta_3 \ne 0 - \text{The effect of temperature on dew point differs by location} \]

All hypothesis tests will use the following level of significance as a guidline for the p-values:

\[ \alpha = .05 \]

We fit the following multiple linear regression model:

\[ \text{DEWP} = \beta_0 + \beta_1 \cdot \text{TEMP} + \beta_2 \cdot \text{NAME}_{\text{Yakima}} + \beta_3 \cdot (\text{TEMP} \cdot \text{NAME}_{\text{Yakima}}) + \varepsilon \]

Where: - \(\beta_0\): Intercept for Rexburg
- \(\beta_1\): Effect of TEMP in Rexburg
- \(\beta_2\): Yakima’s intercept difference from Rexburg
- \(\beta_3\): Yakima’s slope difference from Rexburg

Analysis

To begin the analysis, we will look at a scatterplot to view any possible trends, outliers, skewness of the data.

rexburg <- get_GSOD(years = 2024, station = "726818-94194")
yakima <- get_GSOD(years = 2024, station = "727810-24243")

weather <- rbind(rexburg, yakima)

ggplot(weather, aes(x = TEMP, y = DEWP, color = factor(NAME))) +
  geom_point() +
  geom_smooth(method = "lm", formula = y ~ x) +
  labs(title = "Effect of Temperature on Dew Point",
       x = "Temperature (°F)",
       y = "Dew Point (°F)")

Interestingly, there seems to be no outliers nor filtering that is necessary. In the plot, reveals a seemingly clear linear trend in both cities: as temperature increases, dew point tends to increase. This suggests a positive linear relationship between temperature and dew point in both locations. We can verify this with some model examination, a correlation, and a p-value. Another interesting note, there seems to be no additional insight between the locations Rexburg and Yakima to temperature and dewpoint.

weather.lm <- lm(DEWP ~ TEMP + NAME + TEMP:NAME, data = weather)

  • TEMP (\(\beta_1\)): 0.6083

  • Yakima (\(\beta_2\)): 0.7493

These coefficients suggest that while both cities exhibit a strong positive relationship between temperature and dew point, the slope is shallower for Yakima, and Yakima starts slightly higher.

To estimate the y-intercept for Yakima, we need to add the slope difference, .7493, with the expected value when temp is 0, -4.551. This gives us a y-intercept for Yakima of -3.8017. This number can be interpreted as Estimated dew point in Yakima when temperature is 0°F

In a simular fashion, to find the slope for Yakima directly we just use algebraic manipulation of the slope and slope differences to get .5075. This number can be interpreted as each 1°F increase in temperature is associated with a 0.6083°F increase in dew point for Yakima.

Significant p-values support rejecting all three null hypotheses, as they are all below the threshold of .05. But can we trust these p-values?

Model Diagnostics

To verify model validity, we will examine the residual behavior.

plot(weather.lm, which = 1)

qqPlot(weather.lm$residuals)

## [1] 377 378
plot(weather.lm$residuals, main="Residuals vs. Index")

  • The Residuals vs. Fitted plot shows mild curvature, suggesting some non-linearity.
  • The Q-Q plot indicates that residuals are mostly normal with some tail deviations.
  • The Residuals vs. Index plot shows no obvious pattern, suggesting residual independence.

Because we can still reasonably reject the null hypothesis, we say that temperature does affect the dew point in Rexburg, Yakima has a different baseline dew point than Rexburg, and the effect of temperature on dew point differs among the two locations.

Conclusion

Overall, the linear model appears to fit the data reasonably well, with an adjusted R² of 0.7829 indicating that temperature and location explain a substantial portion of the variability in dew point. However, the presence of mild non-linearity or potential outliers may slightly influence the model’s accuracy and should be further explored using residual plots or transformations. The significant interaction term between temperature and the Yakima location (p = 6.611e-06) suggests that the relationship between temperature and dew point is weaker in Yakima compared to the baseline (Rexburg) location. Specifically, while the baseline (Rexburg) location shows a slope of 0.6083, Yakima’s slope is reduced to approximately 0.5075, indicating a smaller increase in dew point per degree of temperature increase. Additionally, Yakima’s intercept is higher by about 0.749, meaning at zero degrees, Yakima tends to have a higher dew point baseline than the other site. While location contributes a minor shift, temperature remains the dominant predictor, which aligns with physical intuition: warmer air can hold more moisture, thus increasing the dew point.
